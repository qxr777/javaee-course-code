# Getting Started

### Reference Documentation
### 需要先启动eureka-server，eureka-waiter-service
### 从注册中心获得waiter-service服务信息，无需指定主机地址和端口
### @EnableDiscoveryClient 访问注册中心
### @LoadBalanced  开启客户端负载均衡
